=== Лабораторна №1–2: HeatmapUI ===

ФАЙЛИ:
index.html
styles.css
script.js
heatmap.min.js (через CDN)

ЯК ЗАПУСТИТИ:

1. Відкрити index.html у браузері.

2. В правому нижньому куті є маленька кнопка ⚙️
   → натиснути на неї

3. У панелі можна:

   - Start Recording
       Увімкнути/вимкнути запис кліків миші.
       Координати кліків записуються в localStorage.

   - Show Heatmap
       Відтворює всі кліки у вигляді теплової карти.

   - Clear Data
       Очищує всі збережені координати.

4. Теплова карта накладається поверх сторінки
   (контейнер #heatmapContainer).

5. Дані зберігаються у localStorage у ключі "clicks".

Все готово для здачі лабораторної роботи.
